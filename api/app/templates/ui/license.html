Install or replace license

Drop your .lic file onto the area below, or click to pick. The current license is only overwritten after the signature check passes.

How it works

  1. Request a license from info@xenpool.com. We send you a signed .lic file by email.
  2. Upload it here. The signature is checked against the public key baked into this build — if the file is corrupt or from a different issuer, the upload is rejected and your current license stays untouched.
  3. On success, the new license takes effect immediately across all processes (api, worker, beat). No restart needed.
  4. Enterprise features (scheduled orders, deputy support, standalone runbooks, advanced maintenance, …) unlock as soon as the new license is active.
