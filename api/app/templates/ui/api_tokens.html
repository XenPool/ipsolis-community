Retention policy

Slice 1 keeps revoked / expired tokens in the database forever — the audit trail of "we used to have token X" stays intact. Tenants under strict record-retention rules can opt in to hard-delete: rows with revoked_at or expires_at older than the window below are removed by a daily Beat task at 03:15 Europe/Berlin (and by Purge now).
